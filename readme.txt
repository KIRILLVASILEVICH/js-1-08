Это сайт по продаже автомобилей. Он называется AutoScout и состоит всего из 2 страниц - список автомобилей и страница для добавления нового автомобиля. Сайт не доделан. Мы не можем удалить автомобиль из списка и не можем отредактировать информацию по нему. А это нужно. Доделайте, пожалуйста, сайт таким образом, что бы мы могли редактировать информацию по уже добавленному автомобилю, а так же удалять его из списка.
